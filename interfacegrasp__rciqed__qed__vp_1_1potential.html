Detailed Description

Evaluates the single particle matrix elements of an arbitrary (even) potential represented as an array on the GRASP grid in the basis of GRASP orbitals.

Populates matrix with the matrix elements of potential v.

Internally, it uses the QUAD routine to perform the integration on the standard GRASP grid. It will multiply the value of the potential with RP, so the user should not do that themselves (i.e. internally TA ~ V * RP).

Parameters
vAn array containing the potential, represented on the GRASP grid.
matrixAn NW x NW real64 array for storing the matrix elements.
